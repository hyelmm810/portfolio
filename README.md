# Hyerim Hwang Portfolio

A single-page exhibition-style portfolio for Operations PM / 운영 기획자 positioning.

## Structure

- `index.html`: deploy entry file
- `assets/`: image assets used by the portfolio
- `assets/atelier-logo/`: local logo assets for the Atelier section

## Deploy

This project is a static site. It can be deployed directly on Vercel by importing this repository.

No build command is required.
